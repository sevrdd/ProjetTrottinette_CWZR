## Guide d'utilisateur
### Description
Ceci est le guide d'utilisateur. Il contient les branchements du module tableau de bord ainsi qu'une explication complète de son affichage.

Pour retourner au README, cliquer [ici](<../README.md>)

### Branchement

**1.** Alimenter le PCB avec l'alimentation 12V.
 
**2.** Brancher le câble HDMI dans le port HDMI du Raspberry Pi.

### Affichage

**En mode DRIVE :**

![Alt text](AffichageNum.png "Affichage DRIVE")

**1.** Affiche l'état des lumières.
  
**2.** Affiche la température.
  
**3.** Affiche l'état de la batterie.
  
**4.** Affiche le courant des moteurs (A).
  
**5.** Affiche la puissance utilisée.
  
**6.** Affiche le mode d'utilisation (*turtle* ou *rabbit*).
  
**7.** Affiche la vitesse (km/h).

**En mode INIT et ERROR :**

![Alt text](AffichageINIT.png "Affichage INIT")

**1.** Affiche les codes d'erreurs.
  
**2.** Affiche l'état des freins d'urgence.
  
**3.** Affiche les versions du programme.
  
**4.** Affiche l'état des clés.
  
**5.** Affiche le mode.
